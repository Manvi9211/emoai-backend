# EmoAI Backend API

Backend API service for EmoAI — an AI-powered emotional companion platform.

## Features

- FastAPI REST API
- Supabase authentication
- Mood tracking APIs
- Journal APIs
- AI chat integration
- Emotion analysis using Hugging Face inference API

## Tech Stack

- FastAPI
- Python
- Supabase
- OpenRouter API
- Hugging Face Inference API
- Render (Deployment)
